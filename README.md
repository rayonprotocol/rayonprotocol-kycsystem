## Getting Started

### Installing

- install all dependencies

```bash
yarn
```

- Complete `.env.example` file to set enviroment variables used for configuration and rename to `.env`

### Running the tests
```bash
# test with coverage
yarn test 
# open coverage reports
open coverage/index.html
```
